## 操作說明
1.利用計算機內建鍵盤或電腦鍵盤輸入算式
2.按下"="計算

### 運算子、內建函數和按鍵功能
```
+ : 使其連接兩邊相加
- : 使其連接兩邊相減
* : 使其連接兩邊相乘
/ : 使其連接兩邊相除
= : 執行算式
() : 左/右括號(括號內優先運算)
+/- : 在數字最前端輸入，表數字正負
AC : 刪除整行算式
C : 刪除單個運算元/運算子
^ : a^b取a的b次方，若b為0.5則可視為開2次方根
% : 使其連接兩邊取餘
log : 傳回指定底數中()內數字的對數 (需補上右括號)，例:log10代表以10為底
))) : 依照左括號數量自動填補右括號
abs : 傳回()內數值之絕對值 (需補上右括號)
1/x : 傳回()內數值之倒數 (需補上右括號)
X! : 傳回()內數值之乘階 (需補上右括號)
In : 傳回()內數值以自然(底數為e)對數 (需補上右括號)
sin : 傳回()內角度之正弦函數 (需補上右括號)
cos : 傳回()內角度之餘弦函數 (需補上右括號)
tan : 傳回()內角度之正切函數 (需補上右括號)
sinh : 傳回()內角度之雙曲線正弦函數 (需補上右括號)
cosh : 傳回()內角度之雙曲線餘弦函數 (需補上右括號)
tanh : 傳回()內角度之雙曲線正切函數 (需補上右括號)
Asin : 傳回正弦函數(Sine)是指定數字的角 (需補上右括號)
Acos : 傳回餘弦函數(Cosine)是指定數字的角 (需補上右括號)
Atan : 傳回正切函數(Tangent)是指定數字的角 (需補上右括號)
鍵盤輸入 : 切換成用電腦鍵盤輸入之模式
上次刪除 : 復原上次刪除的資料
設定/插入常數 : 設定/插入常數
Rad/Deg : 切換使用角度單位
```

### 運算優先度
括號內算式優先，然後再運算^(次方)，最後先乘除後加減

### 執行算式
用內建鍵盤或電腦鍵盤輸入之算式需符合以下標準才能成功執行
1.運算元、運算子、函數以空白間隔
2.不能引用未定義常數
3.每個左括號都要配上一個右括號
例:
```
5 ^ 2 * π
Asin( sin( Asin( sin( 0.5 )  )  )  ) 
( 7 + 9 )  *  ( 8 - 9 )  / 9
X!( abs( -5 ) )
e ^ π
```

### 常數
常數可自行定義和引用，但常數名不可以內建函數或純數字命名

### 電腦鍵盤輸入
按下"鍵盤輸入"的按鈕後可使用打字的方式輸入算式，輸入完後須要在按下"輸入完成"按鈕後才能執行運算
